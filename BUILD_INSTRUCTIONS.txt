# 📦 JobShield AI - Complete Project Package

## Project Overview

**JobShield AI** is a complete standalone Windows desktop application that detects fake job postings using AI-powered analysis. It includes:
- 🔐 User authentication (login/signup)
- 🤖 AI-powered job analysis using Claude
- 📊 Risk scoring and detailed reports
- 📋 Search history with timestamps
- 🎨 Modern, beautiful UI
- 📦 Single-file EXE distribution

---

## ⚙️ Configuration Options

### Change App Name
Edit `package.json`:
```json
"name": "your-app-name",
"productName": "Your Custom Name",
"version": "1.0.0"
```

### Change Colors/Theme
Edit `src/App.jsx` - search for color values:
- `from-blue-500` → change to other colors
- `bg-gradient-to-r` → modify gradients
- Update Tailwind classes

### Change Window Size
Edit `public/main.js`:
```javascript
width: 1400,  // Change width
height: 900   // Change height
```

### Add Custom Icon
1. Create 256x256 PNG image
2. Convert to ICO format
3. Save as `public/icon.ico`
4. Rebuild with `npm run build-exe`

## 🚨 Troubleshooting

### Problem: "npm: command not found"
**Solution:** Install Node.js from https://nodejs.org/

### Problem: "Module not found"
**Solution:**
```bash
rm -rf node_modules package-lock.json
npm install
```

### Problem: Build fails
**Solution:**
```bash
npm cache clean --force
npm install
npm run build-exe
```

### Problem: App crashes on startup
**Solution:**
- Check internet connection
- Try development mode: `npm start`
- Reinstall Node.js modules

### Problem: "Cannot find API"
**Solution:**
- Ensure internet connection is working
- Check Anthropic API availability
- Try analyzing with shorter job posting

## 🎓 Understanding the Code

### App.jsx Structure
```javascript
// Top-level App component
├── LoginPage Component
│   ├── Login form
│   ├── Sign up form
│   └── Demo credentials
└── JobDetector Component
    ├── Header with controls
    ├── Input textarea
    ├── Analysis results
    └── History sidebar
```

### Key Technologies
- **React 18** - UI framework
- **Electron** - Desktop app wrapper
- **Tailwind CSS** - Styling
- **Claude AI** - Analysis engine
- **Lucide Icons** - Icons

### How Analysis Works
1. User pastes job posting
2. App sends to Claude API
3. Claude analyzes for red flags
4. Results parsed and displayed
5. History automatically saved
